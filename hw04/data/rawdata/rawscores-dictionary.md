## Data `rawscores.csv`

###Here's the description of the R objects in `rawscores.csv`:
The data set rawscores.csv contains the raw scores for a fictitious Stat 133 class. The
columns refer to various types of assingments:
每 homework assignments: columns HW1 to HW9, each 100 pts
每 lab attendance: ATT, number of attended labs (0 to 12)
每 quiz scores:
每 QZ1, 12 pts
每 QZ2, 18 pts
每 QZ3, 20 pts
每 QZ4, 20 pts
每 exam 1: EX1, 80 pts
每 exam 2: EX2, 90 pts
