## Data `cleanscores.csv`

###Here's the description of the R objects in `cleancores.csv`:
The data set cleanscores.csv contains the clean scores for a fictitious Stat 133 class. The
columns refer to various types of assingments:
每 homework assignments: columns HW1 to HW9, each 100 pts
每 lab attendance: ATT, number of attended labs (0 to 12)
每 quiz scores:
每 QZ1, 12 pts
每 QZ2, 18 pts
每 QZ3, 20 pts
每 QZ4, 20 pts
每 exam 1: EX1, 80 pts
每 exam 2: EX2, 90 pts
每 Lab score, 100 pts
每 Homework score, 100 pts
每 Quiz score, 100 pts
每 Test1 score, 100 pts
每 Test 2 score, 100 pts
每 Overall score, 100 pts
